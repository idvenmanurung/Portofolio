Smart Lock Door Using RFID.

This RFID-based smart lock system is built using an Arduino, an RFID reader, a servo motor, and an LCD module. When a registered RFID card is scanned, the Arduino verifies the card's ID, displays an "Access Granted" message on the LCD, and activates the servo motor to unlock the door. If the card is not recognized, the LCD shows "Access Denied" and the door remains locked.

Enose Project Using Raspberry PI.

This eNose (electronic nose) project uses a Raspberry Pi connected to multiple gas sensors (such as MQ2, MQ3, MQ4, and MQ135) to detect and classify different gas patterns. The sensor data is collected in real time, processed using Python scripts, and visualized through a web-based dashboard built with React and Recharts. A machine learning model is trained to recognize specific odors based on the sensor data, enabling applications like food freshness detection or air quality monitoring. This project demonstrates the integration of hardware, data processing, and AI in an IoT-based environmental sensing system.
